# java_tasks-task1407
### Краткое описание логики:  
1. **`try`-блок**:  
   - Лев ищет еду (`"ищет еду"`).  
   - Если еда найдена (`food != null`), выводится ее название (`food.name`).  
   - Лев съедает ее (`"все съел"`).  

2. **`catch`-блок**:  
   - Если `food` равен `null`, возникает `NullPointerException`, и выводится `"ничего не нашел"`.  

3. **`finally`-блок**:  
   - Выполняется **всегда**, независимо от наличия исключения.  
   - Лев ложится спать (`"лег спать"`).  
   - Если еды не было (`food == null`), добавляется `" голодным"`.
